## Create Virtual Environment
 - python -m venv "name of the virtual Environment"
 - Acitavte : 
    - Command Prompt : venv\Scripts\activate.bat
    - PowerShell (Windows) : venv\Scripts\Activate.ps1
    - Bash Command : source venv/bin/activate
